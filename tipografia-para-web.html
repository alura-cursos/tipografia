Conhecendo os diferentes tipos de fonte

Serifa

As serifas são pequenos ornamentos localizados nas extremidades de cada letra que têm como função proporcionar maior fluidez na leitura. Esses detalhesfazem com que uma letra praticamente se conecte a outra.

Sem serifa

Fontes sem a serifa normalmente têm aspecto mais simplificado. Não têm o ornamento que proporciona fluidez, em contrapartida podem tornar o texto mais leve.

Espaçamento fixo

Como o próprio nome já diz, fontes de espaçamento fixo, ou monospace possuem sempre a mesma medida entre os caracteres, ou seja, não possui kerning. O Kerning é um ajuste automático que aproxima caracteres que ficariam visualmente afastados por suas formas.

Cursiva

São fontes que simulam um texto feito a mão, com letras com aspecto caligráfico. A leitura desse tipo de fonte pode variar e, normalmente, não é das mais fáceis – não recomendada para textos de parágrafo.

Fantasia

São as demais fontes que não se encaixam nas outras regras e podem ter diferentes aspectos. Quando utilizadas, normalmente estilizam títulos ou elementos decorativos.

Uso legal, instalação e extensões de fontes

TrueType

Criada nos anos 80 pela Apple e implementada pela Microsoft no Windows 3.1, são fontes altamente escaláveis e leves em conjunto de caracteres. São as fontes mais comuns de entrar, sua extensão é .ttf.

OpenType

As fontes OpenType foram criadas pela Microsoft em 1994 baseada nas fontes TrueType, com a intenção de suportar um maior número de caracteres e outros idiomas. Sua extensão é .otf.

PostScript

Esse tipo de fonte foi criado pela Adobe e é voltada principalmente para impressão, pois possui alta definição e grande compatibilidade com impressoras. Ela é encontrada em grupo de arquivos. No Windows e no Mac as seguintes extensões compõem a fonte: .afm, .pfb e .pfm. Já no Linux, as extensões são: .pfa e .afm

As fontes possuem lincenças

Assim como o uso de imagens, o uso de fontes também possui licenças. Existem fontes pagas, de licença gratuita para uso não-comercial e de licença 100% livre. Como exemplo neste curso, vamos utilizar a fonte Rounded Elegance, que possui licença 100% livre e pode ser encontrada no site dafont.com, uma ótima referência para pesquisar e encontrar fontes.

Como instalar fontes em seu computador

OS (Mac)

Modo 1

  1. Clique duas vezes sobre a fonte que deseja isntalar;
  2. Uma janela se abrirá, clique em instalar fonte.

Modo 2

  1. Abra o aplicativo Catálogo de Fontes (Fontbook);
  2. Arraste as fontes que deseja instalar para lá.;
  3. As fontes que você instalar, podem ser vistas na aba Usuário (User).

Windows (Microsoft)

Modo 1

  1. Clique com o botão direito sobre a fonte desejada;
  2. Selecione a opção instalar.

Modo 2

  1. Acesse c:\Windows\Fonts;
  2. Mova a fonte que deseja instalar para lá.

Ubuntu (Linux)

  1. Acesse o diretório /usr/share/fonts/;
  2. Copie o arquivo da fonte para lá;

Incorporando uma fonte em seu site

Para utiizar uma fonte em uma página da web, é necessário que ela esteja incorporada no site, pois caso o usuário não a tenha em seu computador, a página carregará com outra fonte. Para isso, podemos utilizar uma propriedade CSS, como também podemos incorporar a fonte por meio de um serviço online.

@font-face

A propriedade CSS @font-face permite que adicionemos fontes específicas em nosso site e o usuário poderá ver mesmo que não as tenha em seu computador. Vejamos com a fonte Rounded Elegance.

Acesse o arquivo tipografia.css e adicione o seguinte:

@font-face {
	font-family: Rounded_Elegance;
	src: url('../fonts/Rounded_Elegance.ttf');
}

Agora, vamos estilizar todos os títulos de peso 1, adicione também:

h1, h2, h3, h4, nav li {
    font-family: Rounded_Elegance, Helvetica, Arial, sans-serif;
    font-weight: normal;
}
